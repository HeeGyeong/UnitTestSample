# Unit Test Sample

Mockito, Truth, Espresso를 사용한 가장 기본적인 Unit, Ui Test Sample Code 입니다.

주석을 통하여 각 코드의 동작을 작성해두었으며,
기본적인 Test Code를 적용하면서 발생한 오류와 해결 방안 또한 주석으로 작성해두었습니다.
